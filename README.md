# MERN-Chat

Send and receive messages containing text, images, videos and audios.

## Library used

### Front-end

-   TypeScript
-   ReactJS
-   React-icons
-   React-router-dom v6
-   TailwindCSS
-   Framer-motion
-   Socket.io

### Back-end

-   Mongoose
-   Express
-   MongoDB
-   Cloudinary
-   Socket.io
-   JsonWebToken
-   bcrypt
